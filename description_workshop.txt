[h1]Overview[/h1]
Features includes these abilities:
 * A decompiler used to decompile TMOD files.
 * A command console used to execute commands like help.
 * A packet spy utility used to see what packet is coming in.
 * A debugger bar to see RAM usage.
 * A stats bar to see stats like damage multipliers
 * A quick launch tModLoader utility that allows you to launch multiple tModLoader instances
 * A documentation page for every feature!
 * The ability to run JavaScript and Lua right in tModLoader
[h1]Upcoming Features[/h1]
 * A console chat feature that allows you to chat directly from the debug console
 * An asset extractor used to inspect tMod files
