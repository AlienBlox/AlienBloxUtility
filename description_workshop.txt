[h1]Overview[/h1]
Features includes these abilities:
 * A decompiler used to decompile TMOD files.
 * A command console used to execute commands like help.
 * A packet spy utility used to see what packet is coming in.
 * A debugger bar to see RAM usage.
 * A stats bar to see stats like damage multipliers
 * A quick launch tModLoader utility that allows you to launch multiple tModLoader instances
 * A documentation page for every feature!
 * The ability to run JavaScript and Lua right in tModLoader
 * An asset extractor used to inspect tMod files (Right-click to decompile, left-click to inspect and middle click for build.txt of the mod.)
 * A toolbar that grants features like immortality, super flight, town NPC immortality, hitbox seeing, black hole and scripting
 * The ability to quick butcher entities
 * The ability to summon projectiles, NPCs and give buffs (a command.)
 * A Spawning Menu that allows you to spawn Items, NPCs, give Buffs, place Tiles, Tile Entities and Walls
 * A feature that allows you to grab and move NPCs
 * A freeze utility that allows you to freeze NPCs / Projectiles in place
